#### Public

#### Admin

- `POST /admin/login` – Submit credentials, set session (or return JWT for AJAX).
- `GET  /admin/dashboard` – Admin dashboard with stats and quick links.
- `GET  /admin/doctors` – List all doctors (searchable).
- `GET  /admin/doctors/add` – Form to add a new doctor.
- `POST /admin/doctors` – Save new doctor (creates User + Doctor).
- `GET  /admin/doctors/edit/{id}` – Edit form.
- `POST /admin/doctors/edit/{id}` – Update doctor.
- `POST /admin/doctors/revoke/{id}` – Mark doctor as REVOKED.

- `GET  /admin/logout` – Logout.

### REST API (for potential mobile app)

#### Public
- `GET  /api/public/search?q={query}` – Returns JSON list of matching doctors (limited fields).
- `GET  /api/public/doctor/{licenseNumber}` – Returns full doctor details (including verification status).

#### Admin (requires Bearer token)
- `POST /api/admin/auth/login` – Login, returns JWT.
- `GET  /api/admin/doctors` – List all doctors (paginated).
- `POST /api/admin/doctors` – Create new doctor.
- `PUT  /api/admin/doctors/{id}` – Update doctor.
- `DELETE /api/admin/doctors/{id}` – Soft delete (revoke).
